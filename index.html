🤖 Токен Telegram бота
1. Создайте бота через @BotFather → команда /newbot
2. Скопируйте токен вида 1234567890:ABCdef... и вставьте ниже
3. Ученик переходит к боту, пишет /start, получает свой Chat ID
4. Ученик вводит Chat ID в разделе «Уведомления» — и всё работает
Токен хранится только в вашем браузере
